# NCERT Books & Solutions - Landing Page

Welcome to the official landing page for the **All NCERT Books & Solutions** mobile application. This repository contains the static website showcasing the app's features, capabilities, and resources for students across classes 1-12.

## 📱 About the App

**All NCERT Books & Solutions** is a comprehensive educational platform designed to help students excel in their studies. The app provides complete access to NCERT textbooks, detailed solutions, and reference materials with both offline and online capabilities.

### Key Features

- **Complete NCERT Books**: Access all NCERT textbooks for classes 1-12 with offline reading capability
- **Comprehensive Solutions**: Detailed solutions for NCERT, RD Sharma, RS Aggarwal, Lakhmir Singh & Manjit Kaur, and HC Verma
- **Video Tutorials**: YouTube video tutorials available for all classes to enhance learning experience
- **All Subjects Covered**: Mathematics, Physics, Chemistry, Biology, English, Hindi, Social Science, Economics, Accountancy, Political Science, Psychology, and Environmental Studies
- **Exam Preparation**: Resources for CBSE Board, JEE, NEET, UPSC, SSC, Banking, CTET, UGC NET, Railway exams, and more
- **Offline Access**: Download content and study anywhere, anytime without internet connection
- **Dark Theme Support**: Modern UI with dark theme option for comfortable reading
- **Sample Papers**: Board exam sample papers and previous year question papers
- **Dictionary**: Built-in dictionary for quick reference
